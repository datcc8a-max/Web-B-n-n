## 📝 Ghi Chú Quan Trọng

1. **Nguồn:** Tất cả hình ảnh từ [Unsplash](https://unsplash.com) - Miễn phí sử dụng
2. **Kích thước:** Tối ưu hóa cho web (w=500, h=400)
3. **Tự do sử dụng:** Bạn có thể sử dụng, sửa đổi hoặc tải xuống mà không cần phép
4. **Thay thế hình ảnh:** 
   - Chỉ cần thay đổi URL trong `index.html`
   - Hoặc tải hình ảnh của riêng bạn và lưu cục bộ

---

## 🔄 Cách Cập Nhật Hình Ảnh

### Option 1: Sử dụng URL từ Unsplash
```html
<img src="https://images.unsplash.com/[URL]?w=500&h=400&fit=crop" alt="Tên món ăn">
```

### Option 2: Sử dụng hình ảnh cục bộ
1. Tạo thư mục `images` trong dự án
2. Lưu hình ảnh vào thư mục đó
3. Cập nhật URL:
```html
<img src="images/com-tam.jpg" alt="Cơm Tấm">
```
